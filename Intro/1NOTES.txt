# L4 - more Tags 

1. image - <img src="path/to/image" alt="text to be displayed when the server cant display the image">

2. tables - <table>                     // Declaring table element
                <caption>Table Info on Top</caption>
          3.    <tr>                    // declaring table row , means all content will be part of the row
          4.        <th>Column 1</th>   // declaring table headings , will be declared only once on top
                    <th>Column 2</th>    ... 
                </tr>
                <tr>
          5.        <td>Col 1 info</td>  // Declaring table data below table headings , this is the part of data of a row 
                    <td>Col 2 info</td>   ...
                </tr>
            </table>
    
    - Attributes - rowspan , colspan 
            colspan - How many columns [] [] [] will the td or th will take , eg- colspan="2"
            colspan - How many rows [] 
                                    []  will the td or th will take , eg- rowspan="2"
            
            values of rowspan and colspan can not be larger than the table itself in terms of numeric size
