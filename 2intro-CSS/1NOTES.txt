# L1 , L2  - intro

CSS (Cascading Style Sheets)
HTML - meant to create the skeletal structure of the site(skeleton) , CSS however is meant to visually define the website (muscle , skin , hairs)

CSS can be added in 3 ways 
1. <style> body{background-color:red;} </style> - internal  css (placed in <head></head>)
2. <div style="background-color:red; otherstyle=value"> </div> - inline css
3. in any style (.css) file - external css (mainly used)
element1 , element2 {
    properties...
}
properties will be applied to element1 and element2

# L3 - Selectors

div{
    ... will modify all div instances inside the file
}

.class-name{
    ... will modify only the elements with the given class name
}

#id-name{
    ... will moduify only the element with the given ID
}

parent > child{
    ... will modify the only child instances present in the parent tag eg- div > p{...} will modify p only in div
}
(ONLY WORKS ON DIRECT CHILD , NO multi level INHERITANCE)

parent child{
    ... will modify child instances that are present inside another elements in the parent element too , multi level inheritance just with imidiate child
}

*{
    ... universal selector , modifies all elements
}

| --- Pseudo Selector --- |
    a:visited{
        ... link visited
    }
    a:link{
        ... non visited
    }
    a:active{
        ... when we click the element
    }
    a:hover{
        when hovering
    }
    p:first-child{
        ...wherever p is the first child of a parent , then only modify the p
    }
    p:nth-child(3){
        ...applied to p when it is 3rd child of an element
    }
================ THESE WERE MAIN SELECTORS ==================================

# L4 - Box Model - Padding , border , margin

Any element in HTML has - content , padding , Border , Margin , every element is a box
TOTAL height and width will be content + padding + border + margen (all of both sides) , NOT height , width but TOTAL
width or height css properties Take The width of Content in Context , NOT the padding , hence the visual height and width also adds padding to it
IMPORTANT - usually the width ONLY represents the content width HOWEVER
            IF we want the width property to represent the size of content + padding + border , 
            we use --- 
            { 
                height:200px;
                box-sizing:border-box;
            }

Margin Collapse - concept states that IF the margin of two elements overlaps , then the larger margin as applied ONLY once.

#L5 - Fonts , Text , Color Properties

font-family: name1,name2,name3 ;

    Fonts for the text can be changed using this 
        we set multiple names as it is a good practice , so if the first font is not in browser , it looks at second and then third
        usually in the end we add a browser-safe font that can be loaded on any computer.

font-style:italic; 

    We can change the styles of font from italic , oblique and normal 

font-weight :  - is used to set the boldness of the text

    text-decoration : can underline or overline a text
    text-decoration-color : defines the color of the decorations we selected
    text-decoration-style : changes the style of decoration (dotted/dashed/wavey)
    text-decoration-thickness : changes chickness of decoration

External Fonts - Google Fonts - we can import to our code or we can link , 2 possible ways 
                 Import is done in css wheras linking is done in head section of html 
                 Only one of the methods are needed to use the font in our page .

Font Properties - font-size - determines the size of fonts
                  line-height - determines the spacing between the lines
                  letter-spacing - changes the spacing between charecters
                  text-transform - changes the charecters to upper or lowercase

TEXT OVERFLOW - when text takes more space than the container it is held by 
                text-overflow:(ellipses/hidden/clip)
                overflow:hidden;
                word-break:break-all; breaks the word to fill in the container

text-align:(center/left/right/...) - aligns the text inside the container

-- COLORS --
can be represented in - hex ,  rgb , rgba(red , green , blue , alpha) , colorcode , hsl(hue saturation lightness) , hsla(hsl,alpha)
